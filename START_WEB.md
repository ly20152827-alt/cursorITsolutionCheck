# Web应用启动指南

## 🚀 快速启动

### 1. 安装依赖（如果还没安装）

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python run.py
```

### 3. 打开浏览器访问

**http://localhost:8000**

## 📱 功能说明

### 首页
- 系统介绍
- 功能概览
- 快速开始

### 项目管理
- 查看所有项目
- 创建新项目
- 选择项目

### 文档上传与审核
- 上传Word/PDF文档
- 自动解析
- 智能审核
- 查看结果

## 🎯 使用流程

1. **创建项目** → 点击"项目管理" → "新建项目"
2. **上传文档** → 点击"上传文档" → 选择项目 → 选择文件 → 上传
3. **查看结果** → 等待审核完成 → 查看得分和问题 → 查看完整报告

## 🔧 故障排除

### 页面无法访问
- 确认服务已启动（运行 `python run.py`）
- 检查端口8000是否被占用
- 确认访问地址为 http://localhost:8000

### 静态文件加载失败
- 检查 `static/` 目录是否存在
- 确认文件权限正确

### API调用失败
- 检查后端服务日志
- 确认API服务正常运行
- 查看浏览器控制台错误信息
