# Web应用使用指南

## 访问Web界面

启动服务后，在浏览器中访问：

**http://localhost:8000**

## 功能说明

### 1. 首页
- 系统介绍和功能概览
- 快速开始按钮

### 2. 项目管理
- 查看所有项目列表
- 创建新项目
- 选择项目进行文档上传

### 3. 文档上传与审核
- 选择项目
- 上传Word或PDF文档
- 自动解析和审核
- 实时查看审核进度
- 查看审核结果和报告

## 使用流程

### 步骤1：创建项目
1. 点击导航栏"项目管理"
2. 点击"新建项目"按钮
3. 输入项目名称和选择项目类型
4. 点击"创建"

### 步骤2：上传文档
1. 点击导航栏"上传文档"
2. 选择已创建的项目
3. 选择要审核的文档文件（支持.doc, .docx, .pdf）
4. 点击"上传并开始审核"

### 步骤3：查看审核结果
1. 系统自动进行文档解析和审核
2. 实时显示审核进度
3. 审核完成后显示：
   - 审核得分
   - 问题统计（严重问题、一般问题、优化建议）
   - 审核摘要
4. 点击"查看完整报告"查看详细报告

## 界面特性

- **响应式设计**：支持桌面和移动设备
- **实时进度**：显示文档上传、解析、审核的实时进度
- **结果可视化**：直观展示审核得分和问题统计
- **详细报告**：完整的审核报告，包括问题清单和建议

## 浏览器兼容性

推荐使用以下浏览器：
- Chrome（推荐）
- Firefox
- Edge
- Safari

## 注意事项

1. 首次使用需要先创建项目
2. 文档上传后会自动进行解析和审核，请耐心等待
3. 审核时间取决于文档大小和复杂度
4. 建议文档大小不超过50MB

## 故障排除

### 问题：页面无法加载
- 检查服务是否已启动
- 确认访问地址为 http://localhost:8000
- 检查浏览器控制台是否有错误

### 问题：文档上传失败
- 检查文件格式是否支持（.doc, .docx, .pdf）
- 检查文件大小是否超过限制
- 检查网络连接

### 问题：审核进度卡住
- 刷新页面重试
- 检查后端服务日志
- 确认文档格式正确

## API接口

Web界面通过以下API接口与后端交互：

- `GET /api/projects` - 获取项目列表
- `POST /api/projects` - 创建项目
- `POST /api/projects/{id}/documents/upload` - 上传文档
- `POST /api/documents/{id}/parse` - 解析文档
- `POST /api/documents/{id}/review` - 审核文档
- `GET /api/reviews/{id}/report` - 获取审核报告

详细API文档请访问：http://localhost:8000/docs
